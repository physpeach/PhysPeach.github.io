# コンサティーナ
## コンサティーナとは？
コンサティーナ(Concertina)とは、アコーディオンを小さくしたような可愛らしい楽器です。
コンサティーナについて知りたい方、これから始めてみたい方、その他のコンサティーナに関わる全ての人の助けになれば幸いです。
